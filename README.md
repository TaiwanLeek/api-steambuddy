# SteamBuddy

Project to gather useful gaming information from Steam API

## Resources

* Friend list
* Owned game list

## Elements

* Friend list
    * Friend Steam ID
    * Friend since
* Owned game list
    * Game ID
    * Play Time
    * Windows Play Time
    * Mac Play Time
    * Linux Play Time
    * Last played time

## Entities

* Friends (aka friends list of a certain Steam ID)
* Games (aka owned games of a certain Steam ID)
